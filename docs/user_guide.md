# User Guide

<p align="justify">
Welcome to the <strong>GeoE3 User Manual</strong>! This guide will walk you through the essential features and functionalities of GeoE3, ensuring a seamless user experience. Below is an overview of the key sections included in the manual:
</p>

## [Installing GeoE3](https://worldbank.github.io/GEOE3/docs/userguide/install.html)

Step-by-step instructions to help you download, install and configure GeoE3 on your system.

## [Navigation](https://worldbank.github.io/GEOE3/docs/userguide/navigation.html)

Learn how to navigate the plugin efficiently. This section includes:

- **[Setup the project](https://worldbank.github.io/GEOE3/docs/userguide/navigation.html#project-setup)**: A step-by-step guide to help you initialize and configure your project within GeoE3.
- **[Contextual](https://worldbank.github.io/GEOE3/docs/userguide/contextual.html)**: Working under Contextual Dimension.
- **[Accessibility](https://worldbank.github.io/GEOE3/docs/userguide/accessibility.html)**: Processing the Accessibility factors.
- **[Place Characterization](https://worldbank.github.io/GEOE3/docs/userguide/placecharacterization.html)**: Processing the Place Characterization factors.

## [Insights](https://worldbank.github.io/GEOE3/docs/userguide/insights.html)

<p align="justify">
Explore the analytical capabilities of GeoE3, from combining data on the distribution of women of working age, across low, medium, and high population densities with employment opportunities, to deriving meaningful insights to support your work.
</p>

## [Data Collection](https://worldbank.github.io/GEOE3/docs/userguide/datacollection.html)

<p align="justify">
Learn how to efficiently collect, manage and utilize data within GeoE3. This section offers comprehensive guidelines on identifying and sourcing relevant data for use with the GeoE3 tool. It also covers importing, exporting, and handling geospatial data to ensure seamless integration and analysis. To provide practical context, this section includes a detailed **St. Lucia use case**, showcasing real-world applications of data collection and management techniques within GeoE3.
</p>
